# 最终检查清单

## 内容
- [ ] 标题与用户原文一致
- [ ] 顶部说明未被误改
- [ ] 底部结论未被误改
- [ ] 页码正确
- [ ] PCS编号连续、无重复
- [ ] 公式和变量无错误
- [ ] 百分比、单位、频段准确

## 视觉
- [ ] 三栏宽度均衡
- [ ] 栏目标题样式统一
- [ ] 红色只用于问题与风险
- [ ] 无粗红色外围框
- [ ] 图标、设备和波形风格统一
- [ ] 字体大小在PPT中可读
- [ ] 分隔线不抢眼

## 逻辑
- [ ] 每栏都有“原因—机制—后果”
- [ ] 5秒内能看懂页面主旨
- [ ] 图与创新点或挑战文字一一对应
- [ ] 没有与当前版本技术内容冲突的旧元素

## 输出
- [ ] 16:9
- [ ] 分辨率≥1600×900
- [ ] PNG无明显压缩模糊
- [ ] 关键文字建议在PPT中覆盖为可编辑文本
