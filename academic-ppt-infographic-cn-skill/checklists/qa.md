# 最终检查清单

## 内容
- [ ] 标题与用户原文一致
- [ ] 顶部说明未被误改
- [ ] 底部结论未被误改
- [ ] 页码正确
- [ ] PCS编号连续、无重复
- [ ] 公式和变量无错误
- [ ] 百分比、单位、频段准确

## 视觉
- [ ] 三栏宽度均衡
- [ ] 栏目标题样式统一
- [ ] 红色只用于问题与风险
- [ ] 无粗红色外围框
- [ ] 图标、设备和波形风格统一
- [ ] 字体大小在PPT中可读
- [ ] 分隔线不抢眼
- [ ] 每栏有且只有一个明确主视觉，面积足够
- [ ] 没有为装饰而增加的小图标、圆角卡片或重复边框
- [ ] 缩略到25%时，仍能先看到设备/拓扑/问题关系
- [ ] 三栏底部结论共用基线，文字层级一致

## 逻辑
- [ ] 每栏都有“原因—机制—后果”
- [ ] 5秒内能看懂页面主旨
- [ ] 图与创新点或挑战文字一一对应
- [ ] 没有与当前版本技术内容冲突的旧元素
- [ ] 设备、拓扑、控制关系与用户材料一致
- [ ] 定量指标和约束项均有材料依据，没有凭空补充
- [ ] 多机/多簇问题通过设备阵列或关系图体现规模

## 输出
- [ ] 16:9
- [ ] 分辨率≥1600×900
- [ ] PNG无明显压缩模糊
- [ ] 关键文字建议在PPT中覆盖为可编辑文本
- [ ] 参考页与成稿均以同尺寸导出并完成视觉对比
- [ ] 高清位图没有被低分辨率截图替代，关键矢量对象可编辑
