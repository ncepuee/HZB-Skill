# 局部重绘提示词模板

```text
以参考图片为直接编辑目标。

【固定区域】
保持页面顶部标题、顶部说明文字、底部深蓝结论条、页码以及原页面整体比例完全不变。

【修改范围】
只重绘中间的技术图示区域，不修改固定区域，不增加外部粗边框。

【页面风格】
中文科技成果鉴定/科技奖励答辩PPT风格，16:9横版，白色背景；栏目标题优先继承参考页，可采用浅蓝标题带+深蓝字；蓝色表示正常流程，红色表示问题和风险，绿色表示技术措施；采用细灰虚线分隔，学术、正式、工程化、清晰易读。

每栏只设置一个足够大的主视觉。优先使用用户材料中的真实设备图、准确拓扑和实测波形；线路、箭头、流程框、文字和公式使用可编辑矢量对象。减少装饰性小图标、重复圆角卡片和无技术含义边框。

【布局】
中间区域采用三栏结构：
A栏标题：“[标题A]”
B栏标题：“[标题B]”
C栏标题：“[标题C]”

【A栏内容】
- [设备或拓扑]
- [机制1]
- [机制2]
- [机制3]
- 底部总结：“[原因] → [后果]”

【B栏内容】
- [等效电路或控制框图]
- [参数1]
- [参数2]
- [波形/环流箭头]
- 底部总结：“[原因] → [后果]”

【C栏内容】
- [层级流程]
- [多约束图标]
- [执行对象]
- [问题标签]
- 底部总结：“[条件] → [难点]”

【禁止项】
- 不修改顶部和底部内容；
- 不生成红色外围矩形框；
- 不加入与技术无关的装饰；
- 不改变页码；
- 不生成重复或错误PCS编号；
- 不使用海报式大渐变和强阴影。
- 不为追求全矢量而替换准确的工程原图；
- 不凭空增加元件、约束项、指标或百分比。
```
