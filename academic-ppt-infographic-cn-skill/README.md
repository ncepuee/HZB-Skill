# academic-ppt-infographic-cn-skill

用于生成中文科技奖励、成果鉴定、工程技术答辩风格的PPT信息图。

核心入口：`SKILL.md`

配套内容：
- `templates/image_edit_prompt.md`：基于已有PPT截图局部重绘；
- `templates/new_slide_prompt.md`：从零生成新页面；
- `references/style_spec.md`：配色、尺寸、线条与信息密度；
- `examples/challenge_three_column.md`：三栏挑战页实例；
- `checklists/qa.md`：最终质量检查。
