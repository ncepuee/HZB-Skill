# 风格规范速查

## 页面比例
- 16:9
- 推荐生成尺寸：1652×952、1672×941、1920×1080

## 边距
- 左右边距：页面宽度的3%—5%
- 标题区高度：页面高度的10%—14%
- 底部结论条高度：页面高度的8%—11%

## 线条
- 主流程箭头：2—3 px
- 分隔虚线：1—2 px，浅灰
- 模块边框：1—2 px，浅蓝/浅灰
- 禁止使用粗红色外围框，除非用户明确要求标注修改区域

## 图形
- 设备实物图与矢量图可混合，但每栏保持统一
- 等效电路和控制框图优先使用黑/蓝细线
- 红色虚线箭头可用于环流、振荡和异常路径
- 每栏优先设置1个主视觉，占该栏有效内容区约40%—60%
- 真实设备阵列、准确拓扑和实测波形优先于通用图标
- 高清位图负责工程真实性；文字、公式、箭头和强调框保持可编辑
- 不为追求“全矢量”而用通用符号替换更准确的工程原图

## 标题与结论带
- 栏目标题优先继承参考页；可用浅蓝标题带+深蓝字，不强制深蓝圆角条
- 三栏标题应共用基线、高度和字号
- 栏底总结优先使用整栏浅色底带或无框分区，避免卡片套卡片
- 结论文字可分两行：第一行原因/条件，第二行后果/量化指标

## 信息密度
- 每栏3—5个标签
- 每栏1句总结
- 全页不超过3种主色+1种强调色
- 每栏只保留1个主视觉、1个机理层、1个结论层
- 流程框、图标和标签不能同时堆满；图标不增加识别效率时直接删除
- 相似标签使用平行短语，例如“时间精度不足 / 相位不一致 / 载波不一致”
- 缩略到25%时，应先看到设备、拓扑和问题关系，而不是边框

## 工程图片
- 多机/多簇问题使用横向设备阵列或场站实物图体现规模
- 概念波形可矢量绘制；实测波形使用白底或透明背景高清图
- 位图在PPT中的有效分辨率建议不低于150 ppi，避免二次截图
- 拓扑图应裁掉无关留白，保留元件、连接关系和必要变量
