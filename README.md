# HZB-SKill

HZB's custom AI agent skills collection for Claude Code, Codex, and other AI coding tools.

## Skills

| Skill | Description | Status |
|-------|-------------|--------|
| `multivariable-control` | Multivariable feedback control (Skogestad & Postlethwaite): SVD, RGA, H∞, μ-analysis, robust control design | ✅ Complete |

## Usage

Clone this repository to your skills directory:

```bash
# For Claude Code
git clone https://github.com/ncepuee/HZB-SKill.git ~/.claude/skills/HZB-SKill

# For Amp / OpenClaw
git clone https://github.com/ncepuee/HZB-SKill.git ~/.config/agents/skills/HZB-SKill
```
