# 控制之美 第1卷 Skill

Expert knowledge from 吴昊天's "控制之美 第1卷：从传递函数到状态空间".

## 1. 系统建模

### 1.1 微分方程到传递函数

**时域微分方程**：
```
a_n*y^(n) + a_(n-1)*y^(n-1) + ... + a_0*y = b_m*u^(m) + ... + b_0*u
```

**拉普拉斯变换**（零初始条件）：
```
G(s) = Y(s)/U(s) = (b_m*s^m + ... + b_0) / (a_n*s^n + ... + a_0)
```

### 1.2 典型环节

| 环节 | 传递函数 | 特点 |
|------|---------|------|
| 比例 | K | 无失真放大 |
| 积分 | K/s | 消除稳态误差，增加相位滞后 |
| 微分 | Ks | 提前预测，放大噪声 |
| 惯性 | K/(Ts+1) | 一阶滞后 |
| 振荡 | K/(s²/ωn² + 2ζs/ωn + 1) | 二阶系统，可能振荡 |
| 纯滞后 | e^(-τs) | 采样/传输延迟 |

### 1.3 方框图与信号流图

**Mason 公式**：
```
P_k = Σ P_k * Δ_k / Δ
```
- Δ = 1 - ΣL_i + ΣL_iL_j - ... (特征式)
- P_k: 第k条前向通路增益
- Δ_k: 去除与第k条前向通路接触的回路后的特征式

## 2. 时域分析

### 2.1 一阶系统

**阶跃响应**：y(t) = K(1 - e^(-t/T))
- 时间常数 T：响应到 63.2% 的时间
- 上升时间：t_r ≈ 2.2T
- 调节时间（2%）：t_s ≈ 4T

### 2.2 二阶系统

**标准形式**：G(s) = ωn² / (s² + 2ζωn*s + ωn²)

**关键参数**：
- ζ < 1：欠阻尼（振荡）
- ζ = 1：临界阻尼
- ζ > 1：过阻尼
- 超调量：Mp = e^(-πζ/√(1-ζ²)) × 100%
- 峰值时间：t_p = π / (ωn√(1-ζ²))
- 调节时间（2%）：t_s ≈ 4/(ζωn)

### 2.3 稳态误差

**误差系数**：
- 位置误差：K_p = lim_{s→0} G(s)
- 速度误差：K_v = lim_{s→0} sG(s)
- 加速度误差：K_a = lim_{s→0} s²G(s)

**稳态误差表**（单位反馈）：

| 输入 | 0型 | I型 | II型 |
|------|-----|-----|------|
| 阶跃 | 1/(1+Kp) | 0 | 0 |
| 斜坡 | ∞ | 1/Kv | 0 |
| 加速度 | ∞ | ∞ | 1/Ka |

## 3. 频域分析

### 3.1 Bode 图

**绘制规则**：
- 斜率变化点：每个极点 -20dB/dec，每个零点 +20dB/dec
- 相位：每个极点贡献 -90°，每个零点贡献 +90°
- 转折频率：ω = 1/T（惯性环节）

**最小相位系统**：所有零极点在 LHP，Bode 图唯一确定传递函数

### 3.2 Nyquist 稳定判据

**核心公式**：Z = N + P
- Z: 闭环 RHP 极点数（期望为 0）
- N: Nyquist 曲线绕 (-1, j0) 点的净圈数（逆时针为正）
- P: 开环 RHP 极点数

**稳定条件**：Z = 0 ⟺ N = -P

### 3.3 相位裕度与增益裕度

**增益裕度 GM**：
- 相位穿越频率 ω_π：∠G(jω_π) = -180°
- GM = -20log|G(jω_π)| (dB)

**相位裕度 PM**：
- 增益穿越频率 ω_c：|G(jω_c)| = 1
- PM = 180° + ∠G(jω_c)

**典型设计要求**：PM ≥ 45°，GM ≥ 6dB

## 4. 根轨迹法

### 4.1 绘制规则

**开环传递函数**：KG(s)H(s) = K * N(s)/D(s)

**根轨迹特性**：
1. 起点：K=0 时在开环极点
2. 终点：K→∞ 时在开环零点（无穷远处）
3. 分支数 = max(n_p, n_z)
4. 实轴上的根轨迹：右侧实极点和实零点总数为奇数
5. 渐近线：角度 = (2k+1)×180°/(n_p-n_z)，交点 = (Σp - Σz)/(n_p-n_z)
6. 分离点：dK/ds = 0

### 4.2 设计应用

**主导极点法**：
- 选择期望的闭环极点位置
- 确定所需的根轨迹增益 K
- 如根轨迹不经过期望点，添加零极点校正

## 5. 控制器设计

### 5.1 PID 控制器

```
u(t) = Kp*e(t) + Ki*∫e(t)dt + Kd*de(t)/dt
```

**各环节作用**：
- P（比例）：减小上升时间，减小但不能消除稳态误差
- I（积分）：消除稳态误差，但增加超调和调节时间
- D（微分）：减小超调和调节时间，但放大高频噪声

**PID 整定方法**：
- Ziegler-Nichols 临界比例法
- Cohen-Coon 方法
- 内模控制（IMC）方法

### 5.2 超前-滞后补偿

**超前补偿**（提高相位裕度）：
```
G_c(s) = K_c * (αTs + 1) / (Ts + 1),  α > 1
```
- 最大超前角：φ_max = arcsin((α-1)/(α+1))
- 出现在频率：ω_m = 1/(T√α)

**滞后补偿**（提高低频增益）：
```
G_c(s) = K_c * (βTs + 1) / (Ts + 1),  β < 1
```
- 提高低频增益，减小稳态误差
- 不影响穿越频率附近的相位

### 5.3 根轨迹补偿

- 超前补偿：增加零点，吸引根轨迹向 LHP
- 滞后补偿：增加零极点，改善稳态性能

## 6. 状态空间方法

### 6.1 状态空间表示

```
ẋ = Ax + Bu
y = Cx + Du
```

**传递函数关系**：G(s) = C(sI - A)^(-1)B + D

### 6.2 可控性

**可控性矩阵**：Co = [B AB A²B ... A^(n-1)B]
- 系统完全可控 ⟺ rank(Co) = n

**PBH 检验**：系统完全可控 ⟺ rank[A - λI | B] = n 对所有特征值 λ

### 6.3 可观性

**可观性矩阵**：Ob = [C; CA; CA²; ...; CA^(n-1)]
- 系统完全可观 ⟺ rank(Ob) = n

**对偶关系**：(A,B) 可控 ⟺ (A',C') 可观

### 6.4 状态反馈极点配置

**设计步骤**：
1. 计算特征多项式：det(sI - A) = s^n + a_(n-1)s^(n-1) + ... + a_0
2. 确定期望特征多项式：(s - p_1)(s - p_2)...(s - p_n)
3. 计算增益矩阵 K

**Ackermann 公式**：
```
K = [0 ... 0 1] * Co^(-1) * α_d(A)
```
- α_d(A): 期望特征多项式代入 A

### 6.5 状态观测器

**全阶观测器**：
```
x̂̇ = Ax̂ + Bu + L(y - Cx̂)
```
- 观测器增益 L 使 (A - LC) 稳定

**分离定理**：状态反馈增益 K 和观测器增益 L 可以独立设计

**降阶观测器**：只估计不可直接测量的状态

## 7. 关键公式速查

| 公式 | 表达式 | 用途 |
|------|--------|------|
| 传递函数 | G(s) = Y(s)/U(s) | SISO 系统描述 |
| 特征方程 | 1 + G(s)H(s) = 0 | 闭环极点 |
| Mason 公式 | P = ΣP_kΔ_k/Δ | 信号流图求传递函数 |
| Bode 稳定判据 | PM > 0, GM > 0 | 频域稳定性 |
| Nyquist 判据 | Z = N + P | 频域稳定性 |
| 超调量 | Mp = e^(-πζ/√(1-ζ²)) | 二阶系统性能 |
| 状态反馈 | u = -Kx | 极点配置 |
| Ackermann | K = [0..1]Co^(-1)α_d(A) | 求 K |
| 观测器 | x̂̇ = Ax̂ + Bu + L(y - Cx̂) | 状态估计 |

## 8. MATLAB 常用命令

```matlab
% 传递函数
sys = tf(num, den);
sys = zpk(z, p, k);

% 时域响应
step(sys);       % 阶跃响应
impulse(sys);    % 脉冲响应
lsim(sys, u, t); % 任意输入响应

% 频域分析
bode(sys);       % Bode 图
nyquist(sys);    % Nyquist 图
margin(sys);     % 增益/相位裕度

% 根轨迹
rlocus(sys);     % 根轨迹图
rlocfind(sys);   % 交互式选点

% 状态空间
sys_ss = ss(A, B, C, D);
Co = ctrb(A, B);  % 可控性矩阵
Ob = obsv(A, C);  % 可观性矩阵
K = place(A, B, p);  % 极点配置
L = place(A', C', p)';  % 观测器增益
```
